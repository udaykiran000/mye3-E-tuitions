# Mye3-Elearning Backend

## 🚀 How to Run

Do NOT use `node sever.js` (typo). Use the following:

### 1. Development Mode (Auto-restart)
```bash
npm run dev
```
(This runs `nodemon server.js`)

### 2. Standard Mode
```bash
npm start
```

### 3. Manual Mode
```bash
node server.js
```

## ⚙️ Setup
1. Ensure your `.env` file has the correct `MONGODB_URI` and `JWT_SECRET`.
2. Run `npm install` before starting.
